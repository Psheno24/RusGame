## ФИЛОСОФИЯ

Подработки = старт игры.
Карьера = основа поздней игры.
Игрок может ехать в любой город сразу.
Города не ограничивают игрока искусственно.
Экономика должна сама подсказывать оптимальный путь развития.

Цель:
Омск → Самара → Казань → Екатеринбург → СПб → Москва

Путь должен занимать месяцы активной игры и около года полного развития.
